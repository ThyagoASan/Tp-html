Presentación del sitio

Este sitio web fue desarrollado como actividad práctica de HTML. Está compuesto por cinco páginas vinculadas entre sí mediante enlaces internos.

En las distintas secciones se puede observar el uso de imágenes, tablas, enlaces externos, correo electrónico y un mapa de ubicación.
